css3-animation

不知道大家对它是不是很熟悉了呢
animation: name duration timing-function delay iteration-count direction
animation 属性是一个简写属性，用于设置动画属性：

  • animation-name:动画名称，需要用@keyframes去设置一个动画
  • animation-duration:动画完成的周期，0则不会播放动画
  • animation-timing-function:动画速度的曲线
  • animation-delay:延迟执行动画的时间,也可以负值
  • animation-iteration-count:动画被播放的次数
  • animation-direction:播放方向
  • animation-fill-mode:设置动画结束后的状态
  • animation-play-state:

以上的属性具体的值不再赘述，需要提的一个就是，当我们做雪碧图的逐帧动画的时候我要设置animation-timing-function: step-start这样就会一帧一帧的去显示而不会移动移动雪碧图

不想赘述但是我还是想提几个你可能可能可能不知道的，算我贱得慌好了

  • animation-direction
    reverse:动画反向播放
    alternate:正反正反正反的播放
    alternate-reverse:反正反正反正的播放

  • animation-timing-function
    step-start:按keyframes设置逐帧显示，第一帧为keyframes设置的第一帧
    step-end:按keyframes设置逐帧显示，第一帧为样式的初始值
    steps(n,start|end):把keyframes里设置的一帧等分为几帧.start为第一次显示第一帧，end第一次显示样式的初始值，例如：steps(8,start)
    cubic-bezier(n,n,n,n):在 cubic-bezier 函数中自己的值。可能的值是从 0 到 1 的数值。贝兹曲线限制了首尾两控制点的位置，通过调整中间两控制点的位置可以灵活得到常用的动画效果

  • animation-fill-mode
    forwards:设置对象状态为动画结束时的状态,100%或者to时，也是显示最后动画结束的样子

animation动画是不需要触发的，写在页面就会立即执行，这样对于我们来说是不太好控制的，所以我们用添加class的方式来触发

css3-transition

相信你们对这个也很是了解了，它叫css过渡，通常我们也是通过添加class和改变元素属性来触发
看似很简单的过渡，但是和transform配合可以做出很不错的动画效果来

svg

SVG是一种矢量图格式，具体格式或者嵌到页面是怎么样的请自行查询，下面说一下他这个动画是咋个实现法

<set>实现基本的延迟功能,在这个标签上可以设置
<animate>实现单属性的动画过渡效果，需在标签上设置属性
<animateTransform>这里的transform变换与CSS3的transform变换都是一个路数
<animateMotion>可以让SVG各种图形沿着特定的path路径运动

svg的路径动画需要借助工具将路径的点的相关数据做出来，svg也有很多js类库可以使用，可以说svg在路径动画方面有很深的潜力；后续的可能会专门写一个svg动画的文章，现在还没有深入写过
